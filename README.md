# This is local information regarding the Upstart 200 repo:
Upstart 200 is a Website generated by Omeka. The site will be themed by Design & Discovery at University of Michigan Library. The purpose of the site is to display the digital objects available at the Library.

The following is related to Omeka
Omeka includes:

* [Zend Framework](http://framework.zend.com)
* [getID3](http://getid3.sourceforge.net)
* [jQuery](http://jquery.com)
* [jQuery UI](http://jqueryui.com)
* [TinyMCE](http://tinymce.moxiecode.com)
* [Silk Icons](http://www.famfamfam.com/lab/icons/silk/)

Use and modifications of these libraries must comply with their respective 
licenses.
